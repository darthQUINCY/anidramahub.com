# 🎌 OtakuVerse

A modern, high-performance streaming platform for **Anime, K-Dramas, and C-Dramas** built with a focus on **UX, speed, and conversion**.

---

## 🚀 Features

- 🎥 Trending Anime, K-Drama & C-Drama content
- 🔥 “Top 10 Today” dynamic ranking
- 🤖 AI-based personalized recommendations
- 📺 Smooth video player with autoplay episodes
- 📱 Mobile-first responsive design
- 💾 Watchlist & Continue Watching
- 🌙 Dark theme UI with neon accents

---

## 🧱 Tech Stack

- Frontend: React / Next.js
- Styling: Tailwind CSS
- Backend: Node.js / Firebase / Supabase
- Database: MongoDB / PostgreSQL
- Video: HLS / Streaming APIs
- AI: Recommendation Engine (future integration)
